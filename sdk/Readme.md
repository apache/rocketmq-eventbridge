# Daranbonba

通过 darabonba 生成的多语言 SDK 实现对 eventBridge 进行 RPC 管控

# 多语言 SDK 生成

## dara 安装

```
npm install @darabonba/cli -g
```

# 目录结构

- demo 
    - 包含了 sdk demo 的源码和生成样例，按 controller 分类
- doc
    - 各语言的 sdk 生成使用文档
- generated
    - 包含了 sdk client 的各语言生成样例
- Darafile
    - dara 配置文件
- main.dara
    - dara 源码

# 支持语言

- java
- golang
- csharp
- php
- python
- typescript

# 使用
各语言 sdk 使用方法请参考 doc 中的文档
